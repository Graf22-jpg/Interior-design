# Студія дизайну інтер'єру "INTERIOR"

Курсовий проект з розробки веб-інтерфейсу.

## Функціонал
- **Адаптивна верстка:** сайт коректно відображається на ПК, планшетах та смартфонах.
- **Динамічне портфоліо:** проекти завантажуються за допомогою JavaScript.
- **Валідація форми:** перевірка коректності введення імені, email та повідомлення.
- **Інтерактивність:** мобільне меню (бургер) та плавна навігація.

## Технології
- HTML5, CSS3 (Flexbox, Grid)
- JavaScript (DOM Manipulation, RegExp)
- GitHub Pages для хостингу
