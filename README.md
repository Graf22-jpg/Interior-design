# Курсова робота з Web-дизайну 🚀

Реалізація завдання: адаптивний багатосторінковий вебсайт для студії дизайну інтер'єру **"INTERIOR"**.

## 🔗 Посилання на проект
* **Сайт на GitHub Pages:** [Натисніть тут для перегляду](https://Graf22-jpg.github.io/Interior-design/)

---

## 📋 Виконані етапи практики (згідно з календарним графіком)
1. **Підготовчий етап:** Аналіз аналогів, розробка структури проекту та карти сайту.
2. **Проектування:** Підбір колірної палітри, сучасних шрифтів та оптимізація графічного контенту.
3. **Верстка:** Створення семантичної та адаптивної структури сайту.
4. **Програмування:** Реалізація інтерактивних елементів на чистому JavaScript.
5. **Тестування:** Валідація коду та публікація на GitHub Pages.

---

## 🛠️ Технологічний стек
* **HTML5:** Семантична верстка (`<header>`, `<main>`, `<section>`, `<footer>`).
* **CSS3:** Адаптивна розкладка на базі гнучких сіток **Flexbox** та **CSS Grid**. Дотримано підходу **Mobile First**.
* **JavaScript (ES6+):** Чистий код для логіки сайту (без сторонніх бібліотек).

---

## 🌟 Реалізований функціонал (Рівень І)

Проект повністю задовольняє критеріям найвищого рівня складності та містить такі модулі:
* 📱 **Адаптивне мобільне меню ("burger menu"):** Повністю робоче меню для смартфонів із блокуванням скролу фону.
* 🔄 **Динамічне портфоліо:** Автоматична генерація карток виконаних проектів із масиву об'єктів у JS за допомогою маніпуляцій з DOM.
* ✍️ **Інтерактивна форма відгуків:** Можливість залишати відгуки на сторінці, які на льоту створюються як нові DOM-вузли та додаються вгору списку без перезавантаження сторінки.
* 📝 **Валідація форм:** Перевірка введення даних користувачем (імені, email, довжини повідомлень) перед надсиланням.
* 👁️ **Анімація при скролі сторінки (Reveal):** Плавна поява контенту під час прокрутки сайту, реалізована за допомогою сучасного **Intersection Observer API**.
* 🚫 **Кастомна сторінка 404:** Стилізована сторінка помилки для покращення UX.

---

## 📱 Перевірка адаптивності (Breakpoints)
Сайт коректно відображається на таких роздільних здатностях:
* `320px - 480px` (Мобільні телефони)
* `481px - 768px` (Планшети)
* `769px - 1024px` (Ноутбуки)
* `1025px - 1200px+` (Десктопні монітори)
